Quick Facts

  • Created in early 1990s
  • Invented by Tim Berners-Lee
  • Used to structure webpages
  • HTML5 is current standard

Why It Matters

HTML made it possible to connect documents across the world, which led to the creation of the modern internet.

HTML Timeline

Early 1990s

HTML was created by Tim Berners-Lee to help scientists share documents. The first version was very simple and focused on text and links.

1995 – HTML 2.0

HTML became standardized. This helped ensure that webpages could be displayed consistently across different browsers.

1999 – HTML 4.01

More structure and formatting options were introduced, including better support for layout and styling.

2014 – HTML5

HTML5 introduced semantic tags, audio/video support, and modern features used in today's websites and applications.
