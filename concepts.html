1. Elements

An HTML element is a complete unit of content, such as a paragraph, heading, or link. Elements usually include an opening tag, content, and a closing tag.

2. Tags

Tags are the keywords inside angle brackets that define HTML elements. For example, <p> starts a paragraph and </p> ends it.

3. Attributes

Attributes provide extra information about an element. For example, a link uses the href attribute to define its destination.

4. Nesting

Nesting means placing one element inside another. Correct nesting helps keep an HTML document organized and valid.

5. Semantic HTML

Semantic HTML uses meaningful tags such as <header>, <main>, <section>, and <footer>. These tags make the structure of a page clearer for users, developers, and search engines.

6. Hyperlinks

Hyperlinks connect webpages and make browsing possible. They are one of the most important ideas behind the web.

7. Document Structure

Every HTML page follows a general structure with a doctype, html element, head section, and body section. This structure helps browsers understand and display the page correctly.
