Headings

Heading tags such as <h1> through <h6> organize the content of a page and show the importance of sections.

Paragraphs

The <p> element is used for regular blocks of text. It is one of the most common elements in HTML documents.

Links

The <a> element creates hyperlinks. Links allow users to move between pages, websites, and sections of a document.

Images

The <img> element displays images. It uses attributes like src and alt to define the image source and description.

Lists

HTML supports unordered lists with <ul> and ordered lists with <ol>. Lists are useful for steps, definitions, and grouped information.

Tables

The <table> element organizes data into rows and columns. It is helpful for structured information like schedules, version comparisons, and statistics.
