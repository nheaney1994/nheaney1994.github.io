About Me

My name is Nicholas Heaney, and I am a student in IT 3203. I am learning the basics of web development, including HTML, CSS, and JavaScript.

About This Project

This website was created as part of a class project to present information about HTML. I chose HTML because it is the foundation of webpage structure and one of the most important technologies in web development.

The purpose of this project is to explain what HTML is, how it developed, what common elements it uses, and what key concepts beginners should understand.

Skills Demonstrated

  • Creating multiple linked HTML pages
  • Using an external CSS stylesheet
  • Building a consistent navigation menu
  • Using tables, lists, images, and semantic structure
  • Applying CSS layout and styling

This is a class project for IT 3203.
